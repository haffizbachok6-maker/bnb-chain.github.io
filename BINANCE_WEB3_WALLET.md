# Binance Web3 Wallet

## Setup Guide
1. **Installation**: Download the Binance Web3 Wallet extension from the official website or the Chrome Web Store.
2. **Create Wallet**: Follow the prompts to create a new wallet. Make sure to securely store your recovery phrase.
3. **Connect to Binance Smart Chain**: After installation, connect the wallet to the Binance Smart Chain by selecting it from the network drop-down.

## Features
- **Multi-chain Support**: Supports Binance Smart Chain and Ethereum.
- **DeFi Applications**: Easily access DeFi applications directly from the wallet.
- **Secure transactions**: All transactions are secured using state-of-the-art cryptography.

## Security Information
- **Private Keys**: Your private keys are stored locally and never shared with Binance.
- **Recovery Phrase**: Always backup your recovery phrase to recover your wallet in the future.
- **Regular Updates**: Ensure that your wallet extension is up to date with the latest security patches.
